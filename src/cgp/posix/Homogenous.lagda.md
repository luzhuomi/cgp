This module contains the definition of homogenousity for pdinstances and pdinstance*'s generated by posix pdU[ r , c ] and pdUMany[ r , w ] respectively.

A set of pdisntances, (pdinstance*'s respectively) is said to be homgenous, if a single (partial) derivative (descendants) is  inhabiting in all of them).

In short, pdU and pdUMany from posix partial derivative produces homogenous injections.

This property is required for the POSIX parsing algorithm's correctness proof in (`posix/ExtendedOrder.lagda.md`).
